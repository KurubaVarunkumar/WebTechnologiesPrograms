BOM

BOM is a core of javascrpit. It provides interaction with the browser.BOM stands for browser object model It provides allowing manupulation of brower it self,location, navigation screen and more.

In BOM we have some objects there

1.Window object

Window object is one of the object in BOM. This represents browsers window and contains proporties and methods for intercting with it. BY the using of this we can get window height and width

2.Screen Object

Screen object is a object in BOM.It represents user's screen. It provides information such as height and width of the screen.


Location

location object is a object. It represents the current url of browser

It allows navigation to different URLs

Hstory represents re brower History. It allows manipulation of the brosewr sessin history

Navigation object

Navigation object is a object in BOM. It provides information about browser and user system.Includes proporties such as the user agent,browser name and version
